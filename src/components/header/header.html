Sobre o Projeto Acesso Livre

O projeto Acesso Livre é uma iniciativa dedicada a promover a acessibilidade no IFBA Campus Eunápolis. Nosso objetivo é mapear e avaliar os meios de acessibilidade disponíveis, facilitando a navegação e o acesso para todos os usuários.

Com este mapa interativo, você pode explorar os locais do campus, verificar avaliações de acessibilidade e contribuir com suas próprias experiências para melhorar a inclusão.
